A supervised-learning project that predicts wine quality scores from physicochemical features such as acidity and alcohol content, for both red and white varietals.

## Tech stack

- **Language:** Python (Jupyter)
- **Libraries:** scikit-learn, NumPy, Pandas, Matplotlib / Seaborn

## Highlights

- **Multinomial logistic regression** for 0–10 quality classification.
- Dataset preprocessing, feature scaling, and exploratory data analysis.
- Separate red and white wine datasets for multi-variant analysis.
