A lightweight Flutter app that generates random dice rolls with a single tap, built as a learning project to practice state management, UI updates, and randomization across Android and iOS.

## Highlights

- Cross-platform mobile app built with Flutter
- Demonstrates fundamental state-management and UI-update patterns
- Clean UI with instant random number generation
- A focused learning project for mobile development basics
